# 🎙️ Voz Interativa com Python + ChatGPT

Este projeto demonstra como integrar **Python**, **JavaScript** e **APIs de Inteligência Artificial** para criar um fluxo completo de interação por voz.  
A aplicação permite que o usuário fale diretamente no microfone, tenha sua fala transcrita em texto, receba uma resposta inteligente do ChatGPT e ouça essa resposta sintetizada em voz.

---

## 📌 Objetivos
- **[Gravação de áudio](ca://s?q=Grava%C3%A7%C3%A3o_de_%C3%A1udio_com_Python)**: capturar a voz do usuário diretamente no navegador.  
- **[Reconhecimento de fala](ca://s?q=Reconhecimento_de_fala_com_Whisper)**: transformar áudio em texto com alta precisão usando Whisper.  
- **[Integração com ChatGPT](ca://s?q=Integra%C3%A7%C3%A3o_com_API_do_ChatGPT)**: gerar respostas inteligentes e contextuais.  
- **[Síntese de voz](ca://s?q=S%C3%ADntese_de_voz_com_gTTS)**: converter texto em áudio para uma experiência interativa completa.  

---

## 🚀 Fluxo do Projeto

1. **Gravação de Áudio**  
   - O usuário fala diretamente no microfone.  
   - O áudio é capturado via **JavaScript** e processado em **Python**.  

2. **Reconhecimento de Fala (Whisper)**  
   - O áudio é transcrito em texto.  
   - O modelo Whisper da OpenAI garante alta precisão, mesmo em diferentes idiomas.  

3. **Integração com ChatGPT**  
   - O texto transcrito é enviado para o modelo GPT-3.5.  
   - O ChatGPT gera uma resposta contextual e coerente.  

4. **Síntese de Voz (gTTS)**  
   - A resposta é convertida em áudio usando **Google Text-to-Speech (gTTS)**.  
   - O usuário ouve a resposta automaticamente.
